# Game Theory Designer

A Next.js prototype for turning messy negotiation context into a clean game-theory frame: players, strategic moves, payoff maps, bargaining zones, likely paths, outcomes, and assumptions to validate.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- Text prompts drive the current heuristic game generator.
- PDF uploads are accepted as context signals; full PDF text extraction is the next integration step.
- The app is structured for GitHub and deployment as a normal Next.js project.
